You are analyzing a sequence of frames from a LEGO assembly video to understand what action is being performed.

**SEQUENCE STRUCTURE:**
You will receive frames in this order:
1. PREVIOUS PLACEMENT FRAME: The last accepted placement state (hands clear, part placed)
2. ACTION FRAMES: ALL frames showing hands actively manipulating parts
3. CURRENT PLACEMENT FRAME: The current candidate placement state (hands clear, part placed)

**YOUR TASK:**

Analyze the action sequence to determine:
1. What action is being performed?
2. Is a new part being added, or is an existing part being adjusted?
3. What part appears to be involved (from action frames)?
4. What is the action narrative (describe what's happening)?

**ANALYSIS GUIDELINES:**

**Action Understanding:**
- Look at ALL action frames to understand the complete action
- If hands pick up a NEW part from outside the assembly → NEW placement
- If hands adjust/press an EXISTING part already on assembly → ADJUSTMENT (duplicate)
- If NO action frames between placements → likely DUPLICATE

**Part Identification (from action frames):**
- Describe what you see in the hands during action frames
- COLOR + TYPE + approximate SIZE (e.g., "dark grey brick, appears to be 2x2")
- Do NOT rely on final placement view - focus on what's in the hands during actions

**Note - Size Determination Using Placement Context:**
- Some LEGO pieces cannot be sized by counting visible studs:
  - **Smooth-top pieces (tiles)**: No visible studs on top
  - **Pieces with mismatched geometry**: Top stud count differs from bottom (e.g., brackets, inverted slopes)
- For these pieces, determine size using **PLACEMENT CONTEXT**:
  1. **Identify the surface underneath** - What brick/baseplate is it being placed on?
  2. **Measure relative coverage** - How much of the underlying surface does it cover?
  3. **Compare to adjacent pieces** - Use nearby parts as size reference
- If placement context is unclear or insufficient, return **"undetermined"** for the size rather than guessing

**Action Types:**
- "pickup_new_part" - Hands pick up a new part from outside assembly
- "place_new_part" - Hands place a new part onto assembly
- "adjust_existing" - Hands adjust/press/reposition existing part
- "no_action" - No meaningful action detected

**BOUNDING BOX (box_2d):**
