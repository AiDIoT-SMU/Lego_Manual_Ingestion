You are analyzing SAM3-segmented LEGO assembly images to identify what changed between them.

**INPUTS:**
1. PREVIOUS PLACEMENT (SAM3 segmented): Assembly cropped on white background (may be None for first placement)
2. CURRENT PLACEMENT (SAM3 segmented): Assembly cropped on white background

**YOUR TASK:**

**If PREVIOUS image is provided** (normal case):
Compare the two SAM3-segmented images to determine:
1. Is there a new part added, or is this the same assembly state?
2. If new part added, what is it? (color, type, size, stud count)
3. Where is the new part located?
4. Provide a tight bounding box around the new part in the CURRENT image

**If NO PREVIOUS image** (first placement):
Analyze the CURRENT image to determine:
1. What part(s) are visible in the first placement?
2. Identify the part (color, type, size, stud count)
3. Where is the part located in the frame?
4. Provide a tight bounding box around the part in the CURRENT image

**ANALYSIS GUIDELINES:**

**First Placement Detection:**
- If NO PREVIOUS image is provided → this is the FIRST placement
- For first placement:
  - Set is_duplicate = false
  - Set has_new_part = true
  - Analyze CURRENT image to identify the initial part(s)
  - Usually this is a baseplate or foundation piece
  - Provide bounding box for the main part visible

**Visual Comparison (when PREVIOUS exists):**
- These are SAM3-segmented images with white backgrounds (noise removed)
- Compare the number of visible parts: prev vs current
- Look for ANY difference in the assembly structure
- The new part will be visible in CURRENT but not in PREVIOUS

**Part Identification:**
- **CRITICAL**: COUNT THE STUDS (round bumps on top) to determine size
  - 1 stud = 1x1
  - 2 studs in a row = 1x2 (rectangular)
  - 4 studs in a 2x2 grid = 2x2 (square)
  - 8 studs in a 2x4 grid = 2x4 (rectangular)
- Identify COLOR by looking at the part
- Identify TYPE: brick (thick), plate (thin), tile (no studs), special piece
- Do NOT guess - only report what you can clearly see

**Spatial Positioning:**
- Location: Where on the assembly (e.g., "left side", "center", "top")
- Reference: What is it attached to? (e.g., "baseplate", "previous white brick")
- Orientation: How is it oriented? (e.g., "horizontal", "vertical", "upright")

**Bounding Box (box_2d):**
- Format: [ymin, xmin, ymax, xmax] where each value is 0-1000
- 0 = top/left edge, 1000 = bottom/right edge
- Draw a TIGHT box around the newly added part in CURRENT image
- If no new part, set box_2d to null

**Duplicate Detection:**
- If prev and current look identical → is_duplicate = true
- If you see the same number of parts in same positions → is_duplicate = true
- Only mark as new part if you clearly see an additional piece

**OUTPUT FORMAT:**

Return ONLY valid JSON:

```json

  "has_new_part": true,

  },
  "spatial_position": {
    "location": "left side",
    "reference_object": "grey baseplate",
    "orientation": "upright"
  },
  "box_2d": [400, 100, 650, 300],
  "confidence": 0.95,
  "reasoning": "PREVIOUS image shows baseplate with 2 white bricks. CURRENT image shows baseplate with 2 white bricks PLUS one additional dark grey brick on the left side. STUD COUNT: The new dark grey brick has 4 visible studs arranged in a 2x2 grid (square pattern), confirming it is a 2x2 brick, not 1x2. The assembly grew from 2 parts to 3 parts."
}
```
