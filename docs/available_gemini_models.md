## Generative Models for VLM Tasks

These models can be used for vision-language tasks. All Gemini models (except embeddings/TTS) support multimodal inputs including images.

## Configuration

To use a different model, update your `.env` file:

```bash
# Recommended models (as of script run):
VLM_MODEL=gemini/gemini-2.5-flash       # Stable, balanced performance
VLM_MODEL=gemini/gemini-2.5-pro         # High-quality, more expensive
VLM_MODEL=gemini/gemini-2.5-flash-lite  # Fast, cost-effective
VLM_MODEL=gemini/gemini-robotics-er-1.5-preview  # Specialized for robotics

# Alternative models:
VLM_MODEL=gemini/gemini-3-pro-preview   # Preview of next generation
VLM_MODEL=gemini/gemini-3.1-flash-lite-preview  # Lightweight preview
```

**Note:** Model availability may change over time. Run `python scripts/check_available_gemini_models.py` to get the latest list.
